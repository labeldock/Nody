FIND

select는 css스타일로 노드를 찾습니다. FIND는 루트도 선택될 수 있습니다. 이 부분은 querySelector랑 조금 다른점입니다.


<ul class='list'>
	<li class='first'>first</li>
	<li class='second'>seconde</li>
	<li class='third'>third</li>
</ul>
							

FIND('ul.list li:first-child');
FIND('ul.list li:nth-child(2)');
FIND('li','ul');
FIND('ul','ul');
FIND('.second','ul.list li');
FIND('li',0);
FIND('li',2);
Live run
또 다른 FIND 메소드
  • FINDMEMBER : 같은 래벨상의 노드를 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDIN : 모든 자식의 노드를 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDON : 바로 아래 자식 노드만 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDPARENTS : 모든 부모를 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDPARENT : 바로 상위 부모나 트리의 부모 중 하나를 얻어낼 수 있습니다. (두번째 파라메터 문자등으로 더 상위의 부모를 찾을 수 있음)
  • FINDBEFORE : 선택된 노드의 전 맴버들만 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDAFTER : 선택된 노드의 다음 맴버의 맴버들만 찾습니다. (두번째 파라메터 숫자나 문자등으로 필터링 가능)
  • FINDDOCUMENT : 선택된 노드(IFRAME)의 도큐먼트를 찾습니다.

<ul class='list'>
	<li class='first'><span>first</span></li>
	<li class='second'><span>seconde</span></li>
	<li class='third'><span>third</span></li>
</ul>
<iframe id='frm'><iframe>
							

FINDMEMBER('.first');
FINDIN('.list');
FINDON('.list');
FINDPARENTS('.first span');
FINDPARENTS('.first span','[class]');
FINDPARENT('.first span','ul');
FINDBEFORE('.third');
FINDAFTER('.first');
FINDDOCUMENT('#frm');

MAKE는 일반적으로 간단하게 노드를 만들때 도움을 줍니다. 복잡한 구조일수록 NFTemplate 모듈을 활용하는것이 좋습니다.

NFTemplate

NFTemplate는 template태그를 활용하거나 직접 태그를 입력하여 데이터와 동기화된 노드를 만들 수 있습니다.


<!-- Container -->
<ul id="container"></ul>

<!-- Template -->
<template id="foo-temp">
    <li class="foo">foo</li>
	<li class="bar">bar</li>
	<li class="text-success">fb</li>
</template>
							

/* with template tag */
var tmpTemp = new NFTemplate('#foo-temp');

/* with string */
var strTemp = new NFTemplate(
	'<li class="foo">foo</li>\
	<li class="bar">bar</li>\
	<li class="text-warning">fb</li>'
);

tmpTemp.appendTo('#container');
strTemp.appendTo('#container');
Live run

MVVM (Basic)
  • 다중의 뎁스구조 지원
  • 데이터 모델의 구조를 미리 정의하지 않음
  • 일부만 따로 렌더링 할 수 있음
  • 실시간 바인드 지원

<!-- Container -->
<div id="placeholder-bind1"></div>
<div id="placeholder-bind2"></div>
<div id="placeholder-bind3"></div>
							

/* Original data */ 
var data = [
    {index:"1",value:"one"},
    {index:"2",value:"two"},
    {index:"3",value:"three"}
];

/* Presentation object */ 
var dataContext = new NFDataContext(data);

/* Draw model */
var inputViewModel = new NFViewModel(
	function(){
	    return this.placeholder("ul.top")
	},
	function(){
	    return MAKE("li",MAKE("span",{href:this.value("index")}),this.bind("value","input?text"))
	}
);
var labelViewModel = inputViewModel.clone(undefined,
	function(){
		return MAKE("li",MAKE("span",{href:this.value("index")}),this.bind("value","label.label.label-info"))
	}
);

/* View controller */ 
new NFPresentor("#placeholder-bind1",dataContext,inputViewModel,true);
new NFPresentor("#placeholder-bind2",dataContext,inputViewModel,true);
new NFPresentor("#placeholder-bind3",dataContext,labelViewModel,true);
Live run

ZString (Basic)
\() 구문은 곧바로 값을 환산하여 뽑아준다. 
								
//range param
ZSTRING("Total \\(10~20)%"); //=> "Total 17%"
ZSTRING("Total \\(10~20)%"); //=> "Total 12%"
ZSTRING("Total \\(10~20)%"); //=> "Total 15%"
//choice param
ZSTRING("\\(Boy,Girl) say \\(hello,bye,hi)"); //=> "Boy say hello"
ZSTRING("\\(Boy,Girl) say \\(hello,bye,hi)"); //=> "Girl say hi"
ZSTRING("\\(Boy,Girl) say \\(hello,bye,hi)"); //=> "Boy say bye"

ZString (With param)
